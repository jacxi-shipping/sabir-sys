================================================================================
HOW TO USE
================================================================================

INSTALLATION:
1. python -m venv venv
2. source venv/bin/activate (Windows: venv\Scripts\activate)
3. pip install -r requirements.txt
4. python egg_farm_system/app.py

FIRST TIME:
1. Click "Farm Management" → Add 1-4 farms
2. Create sheds for each farm
3. Add parties (suppliers/customers)
4. Set up feed system (raw materials & formulas)

DAILY USE:
1. Record egg production
2. Issue feed to sheds
3. Record sales/purchases
4. Track expenses
5. View dashboard

REPORTING:
1. Click "Reports"
2. Select report type
3. Generate and view
4. Export to CSV

================================================================================
ACCOUNTING SYSTEM
================================================================================

LEDGER RULES:
- Positive Balance: Party owes us
- Negative Balance: We owe the party
- All transactions auto-posted

AUTO-POSTING:
- Sale: Debit party, Credit sales
- Purchase: Credit party, Debit inventory
- Payment received: Credit party
- Payment paid: Debit party
- Expense: Credit party (if linked)

DUAL CURRENCY:
- Store AFG + USD for every transaction
- Track exchange rate used
- No retroactive recalculation
- Query in either currency
